# Funder Design

## Requirements
1. Users can post post an Idea with payment instruction (like a link or scanner).
2. Users can Approve or Disapprove
3. Users should have a profile where they can edit and update (with emailId, profileLogo with alphabets)
4. Idea(post) should have tags associated with them.
5. Users can search for questions based on keywords, tags, or user profiles.
6. Global feed should show top 10 highest Approved ideas.
7. The system should handle concurrent access and ensure data consistency.
